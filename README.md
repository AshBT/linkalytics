##Getting Started with Linkalytics

Step 1 -- Get dependencies

```git clone git@github.com:qadium-memex/linkalytics.git```

```pip3 install requirements.txt```

```brew install redis```


Step 2 -- Get going

Open two terminal windows, in the first type

```rqworker twitter```

In the second, type

```python main.py```
